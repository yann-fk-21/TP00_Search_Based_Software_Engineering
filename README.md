# TP Search-Based Software Engineering

## Description

Ce projet est l'implémentation des algorithmes de la fiche de TD00 du cours Search-Based Software Engineering (SBSE).

## Prérequis

- Go 1.23.1 ou plus

## Installation

Clonez le dépôt et installez les dépendances :

```bash
git clone https://github.com/yann-fk-21/TP00_Search_Based_Software_Engineering.git
cd TP00_Search_Based_Software_Engineering
```

## Utilisation

Le projet comporte 5 dossiers pour chaque exercices de la fiche de TD nommé exercice_x (x = numéro de l'exercice). Et dans chaque dossier on un fichier exercice_x.go et exercice_x_test.go.
Pour exécuter le projet, utilisez la commande suivante pour chaque exerice:

```bash
cd exercice_x
go test
```

x = numéro de l'exercice.
